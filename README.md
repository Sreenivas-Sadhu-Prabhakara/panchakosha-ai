# Panchakosha AI

**Learn AI from the outside in.** A free, hands-on AI-literacy path in five levels — from your first everyday chatbot prompt to building and training agentic AI.

🌐 **Live:** https://sreenivas-sadhu-prabhakara.github.io/panchakosha-ai/

## The five levels (koshas)

The Vedantic *Pancha Kosha* describes the self as five sheaths, gross to subtle. Learning AI has the same shape — from what you can touch and use, inward to how it works and how to make it.

| # | Kosha | Verb | Course | For |
|---|-------|------|--------|-----|
| 1 | Annamaya | **Use** | AI in your day | everyone |
| 2 | Pranamaya | **Master** | AI at work | professionals & power users |
| 3 | Manomaya | **Understand** | Under the hood | the curious |
| 4 | Vijnanamaya | **Build** | Ship AI apps | developers |
| 5 | Anandamaya | **Innovate** | ML, AI & agents | engineers |

Every level is a self-contained course with **lessons**, a **capstone project**, and a **reference pack** you keep.

## What's inside

- Copyable prompts and runnable code on every page (one-click copy).
- A capstone per level — a real project, with success criteria.
- A reference pack per level — cheat sheets, tool tables, glossary, curated links.
- Local progress tracking (saved in your browser).

## Structure

```
index.html              home — the ladder + Kosha ring
assets/css/kosha.css    the "inner light, in layers" design system
assets/js/kosha.js      copy buttons, nav, progress, scrollspy
level-1/ … level-5/     each: index.html · capstone.html · reference.html
sitemap.xml robots.txt 404.html
```

## Run locally

It's plain static HTML — no build step.

```bash
cd panchakosha-ai
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy

Served via **GitHub Pages** from the `main` branch root. Any push updates the live site.
